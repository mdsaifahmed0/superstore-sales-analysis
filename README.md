# Superstore Sales Analysis

## Tools Used
Python | SQL | Power BI

## Project Overview
Analyzed 9,979 retail sales records to identify revenue trends, 
profit drivers, and loss-making segments.

## Key Findings
- 51% sales growth from 2011 to 2014
- Tables sub-category: −$16K loss despite $206K revenue due to excessive discounting
- Negative correlation (−0.22) between discount and profit
- Top customer Sean Miller: $25K sales but −$1,980 net loss
- Overall Profit Margin: 12.54%

## Files
- `superstore_analysis.ipynb` — Python EDA (Pandas, Matplotlib, Seaborn)
- `queries.sql` — 12+ MySQL queries for segmentation & trend analysis
- `dashboard.pbix` — Interactive Power BI Dashboard
